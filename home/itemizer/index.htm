Creating CDASH Document Items

Document Items are a fundamental building block of the CDASH archive. A document Item has two main parts:

  1. Electronic Media -- such as scannned pages.
  2. Metadata Descriptive information about the document used for catalogging, discovery and knowledge preservation.

Each document item is related to a Place Item. A place item may invo;lve s single image thubnail, but the metadata is concerned with a place and its history.

Document and Place items each have their own distinct set of metadata headings and values defined with Omeka Resource Templates with a little help from the Custom Ontology Module.
